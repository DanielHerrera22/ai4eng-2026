La Pregunta / El Reto de Negocio "Imagina que eres el analista de datos de un centro comercial. Tienes una tabla con los ingresos anuales y la puntuación de gasto de miles de clientes, pero no sabes cómo clasificarlos. Tu objetivo es descubrir grupos naturales entre ellos para lanzar campañas de marketing personalizadas. Escribe una función en Python que reciba estos datos, aplique el algoritmo K-Means para dividirlos en un número específico de grupos (clusters), y devuelva a qué grupo pertenece cada cliente junto con las coordenadas exactas del centro de cada grupo."

Especificaciones de la Función Para resolver este problema de segmentación, la función debe tener la siguiente estructura:

Nombre de la función: segmentar_clientes_kmeans

Argumentos que debe recibir:

datos (pandas.DataFrame o numpy.ndarray): Una matriz o tabla puramente numérica que contiene las características de los clientes (por ejemplo, dos columnas: "Ingreso_Anual" y "Puntuacion_Gasto").

num_clusters (int): Un número entero que define en cuántos grupos distintos queremos dividir a los clientes (el valor de k).

Tipo de dato que debe retornar: tuple (conteniendo dos numpy.ndarray).

Qué se espera que retorne explícitamente: La función debe devolver estrictamente una tupla con dos elementos en el siguiente orden:

Etiquetas de los grupos (numpy.ndarray): Un arreglo unidimensional (lista de números) de la misma longitud que los datos originales. Cada número representará el grupo (ej. 0, 1, 2...) que el algoritmo le asignó a ese cliente en particular.

Centroides (numpy.ndarray): Un arreglo bidimensional que contenga las coordenadas matemáticas del "centro" de cada grupo creado. Si pediste 3 clusters y los datos tienen 2 columnas, este arreglo tendrá una forma de 3x2.
